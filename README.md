# My Diary
## Description:
This is a web application to maintain online diary.
## To start:
- Install the dependencies:
    - ` npm i`
 - start the application using
   - `npm start`
## Usage:
- To test 
 ` npm test`
 - To test the single file
  ` npx jest filename`
- To find the coverage: ` npx jest --coverage`
